### Requirements

* ruby2.4 and bundler


### How to run the demo ?

1. clone the project by command:
```shell
\> git clone git@github.com:ranxiang/bindo-payment.git
```

2. install necessary gems by command:
```shell
\> bundle
```

3. start the web server by command:
```shell
\>bundle exec rails s
```

4. visit http://localhost:3000 by web browser.
